# Retail-Customer-Insights-Sales-Analytics
Interactive retail analytics dashboard built using SQL and Power BI to analyze customer behavior, sales trends, loyalty insights, and product performance.
## Project Overview
This project analyzes retail customer purchasing behavior using Python, PostgreSQL, SQL, and Power BI. The objective is to uncover insights into customer spending patterns, product performance, subscription behavior, loyalty segmentation, and seasonal sales trends to support data-driven business decisions.

The project includes:

- Data cleaning and preprocessing using Python
- Business analysis using SQL queries in PostgreSQL
- Interactive dashboard creation in Power BI
- Customer segmentation and sales trend analysis
- Business recommendations based on analytical insights

## Tools & Technologies
- Python (Pandas, NumPy)
- PostgreSQL
- SQL
- Power BI
- DAX
- Jupyter Notebook

## Dataset Summary
- The dataset used in this project is available publicly on Kaggle:
https://www.kaggle.com/datasets/ayeshasiddiqa123/customer-shopping-behavior-dataset
- Records: 3,900 customer transactions
- Features: 18 columns
- Includes:
  - Customer demographics
  - Purchase behavior
  - Product categories
  - Discounts & subscriptions
  - Review ratings & shipping types
